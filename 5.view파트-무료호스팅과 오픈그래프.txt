   :: 웹호스팅

   - 서버 PC를 여러 사용자들에게 분양/판매
   - 무료 웹호스팅 서비스 검색 & 가입! (다양한 무료호스팅이 있음, 비교해서 선택!)
   ※ 닷홈 신청 ---> http://신청ID.dothome.co.kr  으로 자동생성!
   ※ 무료인 이유?  1) Linux  Server   2) 개인정보-이메일,전화번호 ...광고주 유치
       
   ex> 닷홈 서비스
   1. 웹호스팅 사이트 접속 (각자 검색해보고~)
   2. 웹 회원가입 : ex> abc124
   3. 무료호스팅 신청(메뉴에서)
   4. 무료호스팅 계정 자동생성, 기본 셋팅
      ex>[abcmart].dothome.co.kr
   ※ 요즘은 다들 자동화..원래는 사람이 해야해서, 셋팅비를 지불해야하긴 했음
   5. 제공받은 도메인으로 셋팅완료되면 접속해보기
     http://신청ID.dothome.co.kr
   ※ 보통 웹사이트 회원가입 계정과, 웹호스팅 계정은 같을수도있고 다를수도 있음
  6. 웹서비스로 제공하려는 모든 파일, 폴더 ==> 지금은 SNL 을 완성한 결과,
      FTP Client 도구를 이용해 닷홈에서 부여받은 계정 공간에 직접 전송(=업로드)
     해서 사용자에게 서비스를 할 수 있음.
  ※ 용량제한, 트래픽 제한, 업로드 제한(불법자료)이 있고 이를 어기면 삭제되거나
     제제를 당할 수 있음에 유의!!
  ※ FTP Client도 다양하게 있지만, Open Source 프로그램 FilezillaFTP를 추천함!

  서비스 - 다양, 고객 - 다양 <---> 무료 호스팅 검색!
